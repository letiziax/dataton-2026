# dataton-2026
# El costo invisible del cuidado

Dashboard desarrollado para el datatón DAT4ACCIÓN 2026.

## Objetivo
Analizar cómo el trabajo no remunerado impacta la participación laboral femenina.

## Fuentes
- INEGI ENUT 2024
- CEPAL OIG
- ENIGH 2024

## Herramientas
- Python (limpieza)
- Power BI (visualización)

## Resultado
Dashboard con indicadores de carga total, brecha salarial e impacto económico.
